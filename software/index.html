Software

A collection of open source software developed by Spaceman
Python

I have developed a few python scripts to speed up my workflow, they may be handy to some other people so I have published them.

  • Spaceman/qmk-comment-gen
    • A Simple python script which takes a QMK keymap and returns a prettified comment to help other's visualize the keymap.
  • Spaceman/qmk-comment-gen
    • A work in progress text adventure game written in Python and JSON. This was mainly written to help develop my Python skills but contains some useful examples of how to use some complicated features of Python and Pickle.
Guides

A few guides I have created which may be useful to those trying to follow in my footsteps.

  • Spaceman/qmk-newbs-guide
    • A guide to create QMK Keymap and Keyboards written to be easy to understand.
  • Spaceman/AKI-modding-guide
    • A guide for creating C# and JS mods for SPT-AKI an open source modding and singleplayer version of Escape from Tarkov.
