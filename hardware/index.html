Keyboards

All of my open-source keyboards can be found at Spaceman/SpaceboardsHardware. All of the files are provided under CERN OHL. Both the KiCad and Gerber files are uploaded to the repository and updated regularly.

I also have a KiCad Librbay at Spaceman/SpaceLib. Currently it contains a few useful footprints for designing keyboards.

A list of the keyboards that I have helped bring into fruition include:
